# Code Up Level Up 3 demo application

## Requirements

1. Java 8
2. PostgreSQL

## To run the application follow these steps:

1. Create the database scheme 

  ```
  $ createdb culu3
  $ createuser culu3 -P -s
  // with password culu3
  ```

2. Run `./gradlew bootRun`

3. Hack!
